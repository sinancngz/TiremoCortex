LED
HAL (Hardware Abstraction Layer)

Detailed Description

Light Emitting Diode (LED) Controller.

  • Light Emitting Diode (LED) controller drives common anode and cathode of a display device.
  • Common(ICOM) and segment(ISEG) interface the display device mainly.

[LED version per ABOV 32bit chipset]

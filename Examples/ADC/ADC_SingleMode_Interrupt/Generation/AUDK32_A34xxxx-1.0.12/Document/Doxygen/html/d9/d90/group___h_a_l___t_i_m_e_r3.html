TIMER3
HAL (Hardware Abstraction Layer)

Detailed Description

16-bit Multi-PWM channel Timer

  • The source clock prescaled by the dedicated prescaler is fed to Timer3.
  • Timer3 receives the prescaled source clock and produces 16bit counter.
  • Timer3 has three operation modes below.
    1. Interval mode
    2. Back-to-Back mode
    3. Capture mode
  • Moreover, there are the three match data buffers which generate 3-phase complementary PWM signal.

CRC
HAL (Hardware Abstraction Layer)

Detailed Description

Cyclic Redundancy Check.

  • Cyclic Redundancy Check (CRC) generates an error-detection code which is calculated based on polynomials.
  • The error-detection code is used to examine the integrity of data transmission, storage or flash memory.
  • CRC-7, CRC-8, CRC-16 and CRC32 can be available.

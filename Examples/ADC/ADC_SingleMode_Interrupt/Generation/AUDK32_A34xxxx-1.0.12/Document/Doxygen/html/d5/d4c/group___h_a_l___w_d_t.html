WDT
HAL (Hardware Abstraction Layer)

Detailed Description

Watch Dog Timer.

  • A Watch-Dog Timer (WDT) rapidly detects CPU's malfunction such as endless looping caused by noise, and generates WDT signal.
  • The WDT signal can be used for resetting CPU or triggering interrupt.
  • If WDT function is disabled, it can be used as a normal timer to generate an interrupt at a fixed interval.

I2C
HAL (Hardware Abstraction Layer)

Detailed Description

Inter-Integrated Circuit (I2C)

  • Inter-Integrated Circuit (I2C) is a standard protocol using two wires to attach lower-speed peripheral ICs to processors/microcontroller.
  • It supprots transmitting and receiving data with Serial Data Line (SDA) and Serial Clock Line(SCL).

NMI
HPL (Hardware Private Layer)

Detailed Description

Non-maskable Interrupt.

  • A Non-Maskable Interrupt (NMI) is an special system interrupt.
  • The NMI is a capable of generating NMI interrupt instead of normal interrupt of some peripherals.

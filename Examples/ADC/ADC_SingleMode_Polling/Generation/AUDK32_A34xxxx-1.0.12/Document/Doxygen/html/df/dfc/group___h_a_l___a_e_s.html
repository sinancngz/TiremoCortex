AES
HAL (Hardware Abstraction Layer)

Detailed Description

Advanced Encryption Standard (AES)

  • Advanced Encryption Standard (AES) encrypts and decrypts data based on a symmetric-key alogrithm.
  • It is used to protect private or sensitive information.
  • Size of data block and key are 128bit.

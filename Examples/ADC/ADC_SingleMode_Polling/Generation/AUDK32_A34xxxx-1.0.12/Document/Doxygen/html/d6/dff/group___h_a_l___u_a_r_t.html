Detailed Description

Universal Asynchronous Receiver Transmitter (UART)

  • Universal Asynchronous Receiver/Transmitter (UART) is asynchronous serial communication protocol.
  • It is capable of a full-duplex asynchronous serial communication by configuring baud-rate, data bit, parity bit and stop bit.
  • The communiation status is available via line status information such as Parity error, Overrun, Frame error.

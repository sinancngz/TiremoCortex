RTC
HAL (Hardware Abstraction Layer)

Detailed Description

Real-time Clock.

  • Real-Time Clock (RTC) is an internal block that measures the passage of time.
  • It must keep accurate time-of-day.
  • Programmable alarm or automatic wake-up can be implemented by it.

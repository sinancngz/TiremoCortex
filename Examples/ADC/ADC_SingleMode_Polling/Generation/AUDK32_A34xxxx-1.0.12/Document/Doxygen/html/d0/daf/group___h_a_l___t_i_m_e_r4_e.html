TIMER4E
HAL (Hardware Abstraction Layer)

Detailed Description

16-bit Two-Output PWM channel Timer

  • The source clock prescaled by the dedicated prescaler is fed to Timer4E.
  • Timer4E receives the prescaled source clock and produces 16bit counter.
  • Timer4E has three operation modes below.
    1. Interval mode
    2. Back-to-Back mode
    3. Capture mode
    4. One-Shot Interval mode
  • Moreover, there are the two match data buffers which generate 2-phase complementary PWM signal.

TempSens
HAL (Hardware Abstraction Layer)

Detailed Description

Temperature Sensor.

  • Temperature Sensor detects temperature variation by comparing counter of both sensing and reference clock.

[ TempSens version per ABOV 32bit chipset]

ABOV ARM Cortex-M 32bit
A31xxxx series
Cortex-M0+

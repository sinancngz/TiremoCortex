PCU
HAL (Hardware Abstraction Layer)

Detailed Description

Port/GPIO Control Unit.

  • Port Control Unit (PCU) supports the two major functions below.
  • Port Control
    1. Configures an external signal direction
    2. Configures Interrupt trigger mode
    3. Controls Internal pull-up/down and open-drain
  • GPIO
    1. Outputs signal level (H/L) on a pin
    2. Sets an external interrupt mode up (Level, Rising edge, Falling edge, Both edge)
    3. Disables/enables pull-up/down

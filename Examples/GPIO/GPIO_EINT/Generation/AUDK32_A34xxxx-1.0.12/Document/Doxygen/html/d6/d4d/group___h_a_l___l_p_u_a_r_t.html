LPUART
HAL (Hardware Abstraction Layer)

Detailed Description

Low Power Universal Asynchronous Receiver Transmitter (LPUART)

  • Low Power Universal Asynchronous Receiver/Transmitter (LPUART) is asynchronous serial communication protocol.
  • It is capable of a half-duplex and full-duplex asynchronous serial communication by configuring baud-rate, data bit, parity bit and stop bit.
  • The communiation status is available via line status information such as Parity error, Overrun, Frame error.

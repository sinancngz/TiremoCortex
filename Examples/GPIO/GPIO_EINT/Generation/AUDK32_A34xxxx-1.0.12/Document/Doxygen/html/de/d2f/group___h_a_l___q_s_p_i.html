QSPI
HAL (Hardware Abstraction Layer)

Detailed Description

Quad Serial Peripheral Interface.

  • Quad Serial Peripheral Interface (QSPI) is a high-speed serial interface for accessing external flash memory.
  • It supports single, dual, and quad data line modes to improve transfer performance.

AUDK32 Description

Overview

  • AUDK32 stands for "ABOV Unified Development Kit" and reduces an effort, time and cost for development.
  • AUDK32 complys with CMSIS ARM specification basically.
  • It is composed of HAL, HPL, device specific driver and unified configuration system as well.
    • HAL (Hardware Abstraction Layer) is just reference software for customer to understand ABOV MCU and implement their software easily.
    • HPL (Hardware Private Layer) works as adaptation between HAL and device specfic driver.
    • Sub Family Driver is a kind of dedicated softwatre to a certain sub family of ABOV MCU.
    • Unified Configuration System is ABOV property and facility to configure AUDK Package.

Supported Chipsets

  • Cortex-M0/M0+ : A31xxxx series
    • A31G21x, A31G31x, A31G32x, A31G22x, A31G11x, A31G12x, A31T21x, A31L21x, A31L12x, A31L22x, A31G33x, A31S13x, A31G34x, A31C14x, A31C12x
  • Cortex-M3 : A33xxxx series
    • A33G52x, A33M11x, A33G53x
  • Cortex-M4 : A34xxxx series
    • A34M41x, A34M42x, A34L71x

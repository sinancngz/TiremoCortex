# OpenOCD 0.12.0 Pre-Built Package

**Description:**
OpenOCD is a tool that provides on-chip programming and debugging support.

## Project Links

- [Official OpenOCD Repository](https://sourceforge.net/p/openocd/code/ci/master/tree/)

- [Pre-Built Package Repository](https://sourceforge.net/u/abovsemic/openocd/ci/abov32/tree)

## Supported ABOV 32bit Hardware

This package supports the following ABOV 32bit hardware:

- abov32v1 : A33G52x, A33G53x
- abov32v2 : A31G21x, A31G22x, A31G31x, A31G32x, A31T21x
- abov32v3 : A33M11x, A34M41x, A34M42x, A34L71x, A34G43x, A31C14x, A31C12x
- abov32f1 : A31G11x, A31G12x, A31L12x, A31L22x, A31G33x, A31G34x
- abov32f2 : A31L21x, A31S13x

## Build & Usage Instructions

For detailed instructions on building and using this package, please refer to the README file in the [repository](https://sourceforge.net/u/abovsemic/openocd/ci/abov32/tree).

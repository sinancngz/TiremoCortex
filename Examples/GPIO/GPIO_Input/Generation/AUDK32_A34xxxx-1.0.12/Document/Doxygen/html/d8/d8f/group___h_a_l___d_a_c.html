DAC
HAL (Hardware Abstraction Layer)

Detailed Description

N-bits Digital to Analog Converter.

  • Digital-to-Analog Converter (DAC) converts a digital signal into an analog signal.
  • N-bit digital value is transformed to an analog output based on successive approximation logic.

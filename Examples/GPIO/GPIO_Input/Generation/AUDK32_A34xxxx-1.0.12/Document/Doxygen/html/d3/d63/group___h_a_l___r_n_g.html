RNG
HAL (Hardware Abstraction Layer)

Detailed Description

Random Number Generator.

  • Random Number Generator (RNG) generates a random number from on a specific seed.
  • Because a random number has to be unpredicatable, it is used for cryptography function or a variety of security applications.

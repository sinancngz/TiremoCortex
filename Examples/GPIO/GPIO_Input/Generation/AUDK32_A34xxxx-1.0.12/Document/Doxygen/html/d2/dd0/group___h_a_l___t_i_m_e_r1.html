TIMER1
HAL (Hardware Abstraction Layer)

Detailed Description

16-bit General Purpose Timer

  • The source clock prescaled by the dedicated prescaler is fed to Timer1.
  • Timer1 receives the prescaled source clock and produces 16bit counter.
  • Timer1 has four operation modes below.
    1. Periodic mode
    2. Pulse Width Modulation(PWM) mode
    3. One-shot mode
    4. Capture mode

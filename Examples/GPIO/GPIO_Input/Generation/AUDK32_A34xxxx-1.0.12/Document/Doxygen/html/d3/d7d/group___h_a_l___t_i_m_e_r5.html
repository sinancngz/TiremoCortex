TIMER5
HAL (Hardware Abstraction Layer)

Detailed Description

16-bit General Purpose Timer

  • The source clock prescaled by the dedicated prescaler is fed to Timer5.
  • Timer5 receives the prescaled source clock and produces 16bit counter.
  • Timer5 has four operation modes below.
    1. Periodic mode
    2. Pulse Width Modulation(PWM) mode
    3. One-shot mode
    4. Capture mode

[ Timer5 version per ABOV 32bit chipset]

ABOV ARM Cortex-M 32bit
A31xxxx series
Cortex-M0+

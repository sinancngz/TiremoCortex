TIMER6
HAL (Hardware Abstraction Layer)

Detailed Description

16-bit 100Hz General Purpose Timer

  • The source clock prescaled by the dedicated prescaler is fed to Timer6.
  • Timer6 receives 100Hz source clock and produces 16bit counter.
  • Timer6 has an operation mode below.
    1. Periodic mode

[ Timer6 version per ABOV 32bit chipset]

ABOV ARM Cortex-M 32bit

Cortex-M0+

SPI
HAL (Hardware Abstraction Layer)

Detailed Description

Serial Peripheral Interface (SPI)

  • Serial Peripheral Interface (SPI) is a synchronous serial communication interface specification used for short-distance communication using the four wires bleow.
    1. SS : chip select
    2. SCK : serial clock
    3. MOSI : master out slave in
    4. MISO : master in slave out
  • Master/slave mode and full-duplex communication are supported

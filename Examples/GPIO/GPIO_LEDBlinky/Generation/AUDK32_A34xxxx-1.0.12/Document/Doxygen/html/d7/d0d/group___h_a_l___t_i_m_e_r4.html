TIMER4
HAL (Hardware Abstraction Layer)

Detailed Description

16-bit Multi-Input Capture Timer

  • The source clock prescaled by the dedicated prescaler is fed to Timer4.
  • Timer4 receives the prescaled source clock and produces 16bit counter.
  • Timer4 has four operation modes below.
    1. Periodic mode
    2. Pulse Width Modulation(PWM) mode
    3. One-shot mode
    4. Capture mode
  • Moreover, the Timer4 supports XORing input in capture mode.

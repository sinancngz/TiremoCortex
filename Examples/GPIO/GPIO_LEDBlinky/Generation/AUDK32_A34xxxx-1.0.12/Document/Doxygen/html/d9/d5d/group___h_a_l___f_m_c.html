FMC
HAL (Hardware Abstraction Layer)

Detailed Description

Code/Data Flash Memory Controller.

  • Flash Memory Controller (FMC) interfaces an internal code and data flash.
  • It is composed of Code Flash Memory Controller (CFCM) and Data Flash Memory Controller (DFMC) for the interfacees.
  • The internal code/data flash is supposed to store execution code/data binaries basically.
  • For a certain fuctionality, it can access System Flash located at inside of a device.

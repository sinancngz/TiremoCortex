CFMC
HAL (Hardware Abstraction Layer)

Detailed Description

Code Flash Memory Controller.

  • Code Flash Memory Controller (CFMC) is flash memory controller to interface an internal code flash.
  • The internal code flash is supposed to store execution code binaries basically.
  • For a certain fuctionality, it can access System Flash located at inside of a device.

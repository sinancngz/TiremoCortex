SCU_CLK
HAL (Hardware Abstraction Layer)

Detailed Description

Clock sub system in System Control Unit (SCU)

  • A SCU Clock is one of the System Control Unit(SCU) subsystem.
  • This system provides high/low speed clock oscillated by internal oscillator and external oscillator.
  • There is a Phase-Locked Loop (PLL) block that oscillates based on the high speed internal/external clock for more faster system and peripheral operating.
